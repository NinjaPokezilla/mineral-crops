# Mineral Crops

Mod de agricultura de minérios para Minecraft 1.20.4 (Fabric)

## Features

- **12 Plantas de Minérios** que dropam shards (9 shards = 1 minério)
- **4 Tipos de Solo** com bônus de velocidade progressivos
- **Sistema Celestium** - novo minério endgame com ferramentas e armadura
- **Turbo Furnace** - 4x mais rápida, 4 inputs simultâneos
- **Bio-Fuel** - sistema de combustível alternativo
- **Ferramentas** - Harvest Scythe, Watering Can, Seed Bag

## Compilação

O mod é compilado automaticamente via GitHub Actions. Veja a aba "Actions" para baixar o .jar.

### Manual
```bash
./gradlew build
```
